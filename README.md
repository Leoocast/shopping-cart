# Leo Castellanos - Carrito 

Para correr el proyecto solo hay que clonarlo y ejecutar los siguientes comandos dentro de la carpeta.

Descargar las dependencias:
```bash
yarn
```

Correr el proyecto:
```bash
yarn dev
```
